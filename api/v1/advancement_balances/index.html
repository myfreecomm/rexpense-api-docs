Advancement Balances

This section describes the API of the Advancement balances resource. A balance present a balance of the expenses and the advancements of an user in an organization in a currency.

Parameters

amount: float
currency: string
payer: JSON object
receiver: JSON object

Example of Balance

{% highlight javascript %} { "amount": 1256.33, "currency": "USD", "payer": { "id": 1, "name": "Company Inc.", "type": "Organization" }, "receiver": { "id": 1, "name": "John", "type": "User", }, } {% endhighlight %}

List balances

List all balances of the current user logged in the request.

Definition

GET https://app.rexpense.com/api/v1/advancement/balances

Example request

$ curl -u $YOUR_API_TOKEN:X -X GET https://app.rexpense.com/api/v1/advancement/balances \
      -H 'Accept: application/json' \
      -H 'Content-type: application/json'

Response Example Success

HTTP 200 OK
{% highlight javascript %} { "count": 3, "balances": [ { "amount": 956.66, "currency": "JPY", "payer": { "id": 1, ... }, "receiver": { "id": 1, ... } }, ... ] } {% endhighlight %}

List organizations balances

List all balances from the organizations where the current logged in user is Administrator or Manager. The balances collection does not include any information about the current logged in user (this can be achievied in the endpoint above).

Parameters

page: integer
per_page: integer limit to 100

Definition

GET https://app.rexpense.com/api/v1/advancement/balances/organizations

Example request

$ curl -u $YOUR_API_TOKEN:X -X GET https://app.rexpense.com/api/v1/advancement/balances/organizations \
      -H 'Accept: application/json' \
      -H 'Content-type: application/json'

Response Example Success

HTTP 200 OK
{% highlight javascript %} { "count": 3, "current_page": 1, "total_pages": 2, "balances": [ { "amount": 956.66, "currency": "JPY", "payer": { "id": 1, ... }, "receiver": { "id": 1, ... } }, ... ] } {% endhighlight %}

Show balance

Show a balance according to the given user (receiver_id), organization (payer_id) and currency.

Parameters

payer_id: integer required
receiver_id: integer required
currency: string required

Definition

GET https://app.rexpense.com/api/v1/advancement/balances/:payer_id

Example request

$ curl -u $YOUR_API_TOKEN:X -X GET https://app.rexpense.com/api/v1/advancement/balances/1 \
      -H 'Accept: application/json' \
      -H 'Content-type: application/json' \
      -d '{"currency":"USD","receiver_id":1}'

Response Example Success

HTTP 200 OK
{% highlight javascript %} { "amount": 956.66, "currency": "USD", "payer": { "id": 1, ... }, "receiver": { "id": 1, ... } } {% endhighlight %}
