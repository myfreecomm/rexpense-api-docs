Miscellaneous

This section list a set of miscellaneous resources that the Rexpense API supports.

Emojis

The list of supported emojis can be found at http://www.emoji-cheat-sheet.com/. Another good list with comparison between emojis in different platforms and their native Unicode representation is at http://apps.timwhitlock.info/emoji/tables/unicode

To use a emoji, you have to check if the field supports it in Rexpense. If it does, you just have to pass the code of the emoji, in either simple text form or unicode form. For example, the messages Ok, I'll accept this expense :+1: (using simple text) and Ok, I'll accept this expense üëç (using unicode char, may not be visible depending on your browser) will both be rendered as Ok, I'll accept this expense on Rexpense.

Emoji Restriction

Only emoji icons are available. Sounds will not be recognized.
